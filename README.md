### ⚖️ [The TieBreaker](https://github.com/Ahtesham-Latif/Tie-Breaker-App) &nbsp; ![Deployed](https://img.shields.io/badge/Status-Deployed-22c55e?style=flat)
> *A deterministic AI decision engine that transforms "A vs B" dilemmas into structured comparison matrices, SWOT profiles, and context-aware verdicts.*

Built with a strict philosophy: **structure over free-form chat.** The backend enforces JSON schemas that the frontend renders into beautiful, side-by-side dashboards. No chatbot UX — just clean, objective, AI-compiled decision intelligence.

| Layer | Detail |
| :--- | :--- |
| **Frontend** | React 19, TypeScript, Vite, Tailwind CSS v4, Framer Motion |
| **Backend** | Node.js, Express — JWT-verified API gateway with rate limiting |
| **Database** | Supabase (PostgreSQL + GoTrue Auth, RLS row-level security) |
| **AI Engine** | Azure AI Foundry · `gpt-4o-mini` · 0.3 temperature · SSE streaming |
| **Deep Research** | Azure Bing Web Search — live web grounding on demand |
| **Deployment** | Azure App Service (Frontend + Backend) |
| **Security** | Prompt injection prevention via XML tags, IP-based rate limiting |
